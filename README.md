## Features:
* Full CRUD functionalities
* Browse games in "In stock" page
* admin page devoted to handling what games render where (recommended flag)
* Create Account/login/logout functionality
* restriction of page access to non-logged users
* add recommendations for store inventory (dynamically populates "coming soon" page)
* add game, remove game from cart
* simulated checkout
